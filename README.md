这个一个stable-diffusion-webui的插件，用于自定义修改UI颜色
实际上是对gradio库的装饰器做修改，sd-webui界面就是使用的该库
<br>
使用方法：
<br>
到sd扩展页面-从网址安装-输入网页https://github.com/wangdadx/sd_ui_hack.git 即可<br>
